Home Component

This will be a small tutorial of how this was set up...

The home component serves as the default route component.

    src/
     |-app/
       |-home/
         |-home.component.html
         |-home.component.ts

The component will need to be imported into app.component.ts.

    import {HomeComponent} from './home/home.component';

And the route will also have to be added as well.

    @RouteConfig([
      {
        path: '/',
        name: 'Home',
        component: HomeComponent,
        useAsDefault: true
      }
    ])
