About Component

Another component in the application.

src/
 |-app/
   |-about/
     |-about.component.html
     |-about.component.ts

The component will need to be imported into app.component.ts.

import {AboutComponent} from './about/about.component';

And the route will also have to be added as well.

@RouteConfig([
  ...
  {
    path: '/about',
    name: 'About',
    component: AboutComponent
  }
])
